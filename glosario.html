Glosario de Términos

Este glosario reúne términos esenciales relacionados con el mantenimiento preventivo, correctivo y el funcionamiento general de computadoras y software.

A
Actualización

Proceso de instalar versiones más recientes de software o controladores.

Antivirus

Software diseñado para detectar y eliminar malware.

Almacenamiento

Medio donde se guardan datos de forma permanente.

API

Conjunto de funciones que permite la comunicación entre aplicaciones.

Advertencia del Sistema

Notificación sobre fallas o riesgos en el equipo.

B
Backup

Copia de seguridad de datos importantes.

BIOS

Sistema básico de entrada y salida encargado del arranque inicial del equipo.

BSOD

“Blue Screen of Death”, error crítico del sistema operativo Windows.

Bug

Error o falla en el software.

Bandwidth

Cantidad de datos que pueden transferirse por segundo en una red.

C
Cache

Memoria rápida utilizada para acelerar procesos.

Comando

Instrucción que ejecuta una acción en el sistema.

Controlador (Driver)

Software que permite la comunicación entre hardware y sistema operativo.

CPU Usage

Porcentaje de uso del procesador.

Checksum

Método para verificar la integridad de archivos.

D
Diagnóstico

Proceso de identificar fallas en el equipo.

Defragmentación

Organización de archivos en el disco para mejorar el rendimiento.

DNS

Sistema que traduce nombres de dominio a direcciones IP.

Dashboard

Panel visual de control de aplicaciones.

Debugging

Proceso de depuración de errores en el software.

E
Error Crítico

Fallo grave que interrumpe el funcionamiento del sistema.

Especificaciones

Características técnicas del equipo o software.

Extensión

Sufijo que indica el tipo de archivo.

Encriptación

Proceso de proteger datos mediante codificación.

Estabilidad

Grado de buen funcionamiento continuo del sistema.

F
Firmware

Software integrado directamente en el hardware de un dispositivo.

Firewall

Sistema de seguridad que controla el tráfico de red.

Formato

Proceso de borrar y preparar un disco para almacenar datos.

Frecuencia

Velocidad a la que trabaja un procesador o memoria.

FTP

Protocolo utilizado para transferir archivos entre computadoras.

G
Gigabyte

Unidad de almacenamiento equivalente a 1024 megabytes.

GPU

Procesador especializado en gráficos.

Gateway

Dispositivo que conecta diferentes redes.

Gestor de tareas

Herramienta que permite ver procesos y uso de recursos del sistema.

GUI

Interfaz gráfica de usuario que permite interactuar visualmente con el sistema.

H
Hardware

Componentes físicos de un sistema informático.

HDD

Disco duro mecánico utilizado para almacenamiento.

Hosting

Servicio que permite alojar sitios web en servidores.

HTTP

Protocolo utilizado para transferir páginas web.

HTTPS

Versión segura del protocolo HTTP con cifrado.

I
IP

Dirección numérica que identifica un dispositivo en una red.

Interfaz

Medio mediante el cual el usuario interactúa con un sistema.

Instalación

Proceso de agregar software al sistema.

Indexación

Proceso que permite localizar archivos más rápidamente.

Integridad de datos

Garantía de que los datos no han sido alterados.

J
Java

Lenguaje de programación utilizado en múltiples plataformas.

JavaScript

Lenguaje utilizado para agregar interactividad a sitios web.

JSON

Formato de intercambio de datos ligero y fácil de leer.

K
Kilobyte

Unidad de almacenamiento equivalente a 1024 bytes.

Kernel

Núcleo del sistema operativo que controla el hardware.

L
LAN

Red local que conecta dispositivos en un área limitada.

Latencia

Tiempo que tarda un dato en viajar de un punto a otro.

Linux

Sistema operativo de código abierto.

Log

Registro de eventos del sistema.

M
Malware

Software malicioso diseñado para dañar o robar información.

Memoria RAM

Memoria temporal utilizada por el sistema para ejecutar procesos.

Monitorización

Seguimiento del estado y rendimiento de sistemas.

Multitarea

Capacidad del sistema para ejecutar varios procesos simultáneamente.

N
Navegador

Software que permite acceder a páginas web.

Nodo

Punto de conexión dentro de una red.

Nube

Almacenamiento y servicios accesibles a través de internet.

O
Open Source

Software cuyo código fuente es público.

Optimización

Proceso de mejorar el rendimiento del sistema.

Overclocking

Aumento de la velocidad del hardware más allá de sus especificaciones.

P
Ping

Herramienta para comprobar conectividad en red.

Plugin

Extensión que agrega funciones a un programa.

Protocolo

Conjunto de reglas para la comunicación entre dispositivos.

Q
Query

Consulta realizada a una base de datos.

R
Red

Conjunto de dispositivos conectados entre sí.

Router

Dispositivo que dirige el tráfico entre redes.

Resolución

Cantidad de píxeles que muestra una pantalla.

S
Servidor

Equipo que proporciona recursos a otros dispositivos.

Software

Programas y aplicaciones de un sistema.

SSD

Unidad de almacenamiento rápida sin partes mecánicas.

T
TCP/IP

Conjunto de protocolos utilizados en internet.

Terminal

Interfaz de línea de comandos.

Token

Elemento de seguridad utilizado para autenticación.

U
USB

Puerto utilizado para conectar dispositivos.

Usuario

Persona que utiliza un sistema informático.

V
VPN

Red privada virtual que protege la conexión.

Virtualización

Creación de entornos virtuales dentro de un sistema físico.

W
Wi-Fi

Tecnología de conexión inalámbrica a internet.

WAN

Red que cubre áreas geográficas grandes.

X
XML

Lenguaje utilizado para estructurar datos.

Y
Yottabyte

Unidad de almacenamiento extremadamente grande equivalente a 10²⁴ bytes.

Z
Zip

Formato de compresión de archivos.

Zona horaria

Configuración del sistema que define la hora según la región.
