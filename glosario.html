Glosario de Términos

Este glosario reúne términos esenciales relacionados con el mantenimiento preventivo, correctivo y el funcionamiento general de computadoras y software.

A
  • Actualización: Proceso de instalar versiones más recientes de software o controladores.
  • Antivirus: Software diseñado para detectar y eliminar malware.
  • Almacenamiento: Medio donde se guardan datos de forma permanente.
  • API: Conjunto de funciones que permite la comunicación entre aplicaciones.
  • Advertencia del Sistema: Notificación sobre fallas o riesgos en el equipo.
B
  • Backup: Copia de seguridad de datos importantes.
  • BIOS: Sistema básico de entrada/salida encargado del arranque inicial.
  • BSOD: "Blue Screen of Death", error crítico del sistema operativo Windows.
  • Bug: Error o falla en el software.
  • Bandwidth: Cantidad de datos que pueden transferirse por segundo.
C
  • Cache: Memoria rápida utilizada para acelerar procesos.
  • Comando: Instrucción que ejecuta una acción en el sistema.
  • Controlador (Driver): Software que permite la comunicación entre hardware y sistema operativo.
  • CPU Usage: Porcentaje de uso del procesador.
  • Checksum: Verificación de integridad de archivos.
D
  • Diagnóstico: Proceso de identificar fallas en el equipo.
  • Defragmentación: Organización de archivos en el disco para mejorar rendimiento.
  • DNS: Sistema que traduce nombres de dominio a direcciones IP.
  • Dashboard: Panel visual de control de aplicaciones.
  • Debugging: Proceso de depuración de errores en el software.
E
  • Error Crítico: Fallo grave que interrumpe el funcionamiento del sistema.
  • Especificaciones: Características técnicas del equipo o software.
  • Extensión: Sufijo que indica el tipo de archivo.
  • Encriptación: Proceso de proteger datos mediante codificación.
  • Estabilidad: Grado de buen funcionamiento continuo del sistema.
F
  • Firewall: Sistema que controla el tráfico de red para evitar intrusiones.
  • Firmware: Software incrustado en dispositivos electrónicos.
  • Formateo: Proceso de borrar y preparar un disco para su uso.
  • Fragmentación: División de archivos en partes dispersas del disco.
  • Freeware: Software gratuito sin costo de uso.
G
  • Gestor de Tareas: Herramienta para supervisar procesos activos.
  • Gateway: Dispositivo que conecta redes distintas.
  • GPU: Procesador especializado en gráficos.
  • Gigabyte (GB): Unidad de medida de almacenamiento.
  • Glitch: Falla temporal en software o hardware.
H
  • Hardware: Componentes físicos del equipo.
  • HTTP: Protocolo de transferencia de hipertexto.
  • Hertz (Hz): Medida de frecuencia del procesador.
  • Host: Dispositivo que funciona como servidor o nodo.
  • Hotfix: Actualización rápida para corregir errores críticos.
I
  • IP: Dirección que identifica un dispositivo en la red.
  • Interfaz: Medio por el cual el usuario interactúa con el sistema.
  • Instalación: Proceso de agregar software al sistema.
  • Integridad del Sistema: Estado en el que funciona sin alteraciones ni daños.
  • Icono: Representación gráfica de una aplicación o función.
J
  • Java: Lenguaje de programación utilizado en múltiples plataformas.
  • Jumper: Pequeño conector usado para configurar hardware.
  • JPEG: Formato de imagen comprimida.
  • Joystick: Dispositivo de entrada para videojuegos o simulación.
  • JSON: Formato ligero para intercambio de datos.
K
  • Kernel: Núcleo del sistema operativo.
  • Keycap: Cubierta de cada tecla en un teclado.
  • Keyword: Palabra clave en programación o búsquedas.
  • Keylogger: Software que registra pulsaciones de teclado.
  • Kilobyte: Unidad de almacenamiento equivalente a 1024 bytes.
L
  • LAN (Local Area Network): Red de área local.
  • Latency: Tiempo que tarda un paquete de datos en llegar a su destino.
  • Linux: Sistema operativo de código abierto.
  • Logic Board: Placa principal de un dispositivo Apple.
  • Login: Proceso de autenticación del usuario.
M
  • Malware: Software malicioso diseñado para dañar o infiltrarse en sistemas.
  • Motherboard: Tarjeta madre donde se conectan todos los componentes internos.
  • Monitor: Dispositivo de salida que muestra información visual.
  • Megabyte (MB): Unidad de almacenamiento digital.
  • Modem: Dispositivo que permite la conexión a Internet.
N
  • NAS (Network Attached Storage): Sistema de almacenamiento conectado a la red.
  • NIC (Network Interface Card): Tarjeta de red.
  • Notebook: Computador portátil.
  • NTFS: Sistema de archivos usado por Windows.
  • Node: Punto dentro de una red informática.
O
  • Overclocking: Aumentar la velocidad de un componente más allá de su capacidad nominal.
  • OS (Operating System): Sistema operativo.
  • Output: Salida de información generada por el sistema.
  • Open Source: Software de código abierto.
  • OCR (Optical Character Recognition): Reconocimiento óptico de caracteres.
P
  • Procesador: Unidad encargada de ejecutar instrucciones del sistema.
  • PCI Express: Interfaz de conexión para tarjetas de expansión.
  • Patch: Actualización pequeña para corregir errores de software.
  • Power Supply: Fuente de poder.
  • Proxy: Servidor intermediario en una red.
Q
  • Query: Consulta realizada a una base de datos.
  • QoS (Quality of Service): Gestión de prioridad en redes.
  • QR Code: Código de respuesta rápida usado para almacenar datos.
  • Queue: Cola de procesos o tareas en espera.
  • Quick Format: Formateo rápido de unidades de almacenamiento.
R
  • RAM (Random Access Memory): Memoria de acceso rápido usada por el sistema.
  • Router: Dispositivo que gestiona tráfico de red.
  • Reboot: Reiniciar un sistema.
  • Restore Point: Punto de restauración del sistema.
  • Runtime: Tiempo en que un programa está en ejecución.
S
  • Software: Conjunto de programas que permiten el funcionamiento del sistema.
  • SSD (Solid State Drive): Unidad de almacenamiento de estado sólido.
  • Server: Equipo que provee servicios a otros dispositivos.
  • Script: Conjunto de instrucciones automatizadas.
  • Startup: Procesos que se inician junto con el sistema operativo.
T
  • Task Manager: Administrador de tareas del sistema.
  • Terminal: Interfaz de línea de comandos.
  • Trojan: Malware que se oculta dentro de software legítimo.
  • Throughput: Cantidad de datos procesados en un tiempo determinado.
  • Toolkit: Conjunto de herramientas de software.
U
  • USB: Puerto y estándar de conexión para periféricos.
  • User: Persona que utiliza un sistema informático.
  • Update: Actualización de software.
  • Uptime: Tiempo que un sistema lleva encendido sin interrupción.
  • UI (User Interface): Interfaz de usuario de un sistema.
V
  • Virus: Software malicioso capaz de replicarse.
  • VPN (Virtual Private Network): Red privada virtual.
  • VRAM: Memoria de video utilizada por la tarjeta gráfica.
  • Version: Identificador de una edición de software.
  • Vulnerability: Punto débil del sistema o software.
W
  • Wi-Fi: Tecnología de conexión inalámbrica.
  • Windows: Sistema operativo desarrollado por Microsoft.
  • Workflow: Flujo de trabajo en procesos digitales.
  • WAN (Wide Area Network): Red de área amplia.
  • Widget: Elemento gráfico interactivo de una interfaz.
X
  • XML: Lenguaje de marcado para estructurar datos.
  • XAMPP: Paquete de herramientas para desarrollo web.
  • X86: Arquitectura de procesadores.
  • XInput: API usada para controladores de juegos.
  • Xorg: Servidor gráfico para sistemas UNIX.
Y
  • Yottabyte: Unidad de almacenamiento extremadamente grande.
  • YAML: Lenguaje de serialización de datos.
  • Y-axis: Eje vertical utilizado en gráficos.
  • Y-Cable: Cable divisor para dispositivos USB.
  • YouTube: Plataforma usada frecuentemente para tutoriales técnicos.
Z
  • Zip: Formato de compresión de archivos.
  • ZIF (Zero Insertion Force): Tipo de socket para procesadores.
  • Zettabyte: Unidad de almacenamiento de gran escala.
  • Z-buffer: Técnica usada en gráficos 3D.
  • Zone File: Archivo que contiene información DNS.
